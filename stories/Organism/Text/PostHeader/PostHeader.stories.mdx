# Post Header 

The Post Header Component defines a section of information data with some items - Breadcrumb, Header title, Date,  Description. <code>Breadcrumbcomponent</code> and <code>Post</code> Molecules are used for creating this section.

<Story name="Post Header">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <PostHeader data={caption.postheaderdata} header={caption.header} country={caption.country} description={caption.description} active={'default'}></PostHeader>;
  }}
</Story>
