The stats card slider is a type of pattern which uses multiple components and arranges them in a visually appealing way.

The stats card slider is used to show various related statistics.

#### When to use:
- To showcase related stats
- To grab a users attention while scrolling through
- If you have only a small space to display a lot of content.

## Formatting

### Default
<Canvas>
  <Story name="Stats card slider">
    {(args, { globals: { locale } }) => {
      const caption = statsArray(locale);
      return <Statscardslider data={caption}></Statscardslider>;
    }}
  </Story>
</Canvas>

### Content
The stats card slider is made up of a group of stats cards arranged in two rows. The stats cards contain a value, label, and description. The description should be precise and not more than a couple of lines.

The number of cards will be divided equally into two rows and will fit the grid width.

### Behaviors

#### States
A stats card slider has 2 states on larger resolutions. Static and scrolling. 
By default, only the section title of the slider should be displayed. As a user scrolls down the page, the first card from both rows will be shown at the extreme left and right of the page. As the user scrolls further and the slider is in the center of the viewport, all the cards in both rows should be displayed and both rows should be aligned in the center of the page. Additionally, an accent color overlay is displayed on hover

On mobile, this slider is converted to a standard carousel. The card in the center of the viewport will display the hover effect.
