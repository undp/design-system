# Page Hero Component

## Overview

The Page hero component multiusage component which creates boxes that are usually teasing some kind of content like - Image, Text, Subtitle, and Breadcrumb. We are using the animation here assets/js/viewport.js. Atoms tags are used for creating this card.

## Default Representation & various possible states

<div className="subheading">Page Hero</div>

<Story name="Pages Hero">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return (
      <PageHero data={caption.herodata} title={caption.title} content={caption.content} active={'white'} {...args}
      ></PageHero>
    );
  }}
</Story>

<div className="subheading">Page Hero Option</div>

<Story name="Pages Hero Option">
  {(args, { globals: { locale } }) => {

    return (
      <PageHeroOption data={caption.herodata} title={caption.title} content={caption.content} active={'default'} {...args}
      ></PageHeroOption>
    );
  }}
</Story>
