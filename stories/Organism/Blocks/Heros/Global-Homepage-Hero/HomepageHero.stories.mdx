# Homepage Hero Component

## Overview

The Homepage hero component multiusage component which creates boxes that are usually teasing some kind of content like - Image, Text, Subtitle, and CTA Link. We are using the animation here assets/js/viewport.js. Atoms tags are used for creating this card.

## Default Representation & various possible states
