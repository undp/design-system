# Author card component

## Overview

The Author card Component is created for showing the authors name, designation, and image in particular section. It is a multi-usage component that creates are usually teasing some kind of content.

<Story name="Author Summary">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return (
      <AuthorSummary
        image={user}
        para={caption.paragraph}
        text={caption.text1}
        Authorlabel={caption.text2}
        para1={caption.text3}
      ></AuthorSummary>
    );
  }}
</Story>
