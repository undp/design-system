# Content Card Component

The Image Reveal Card component defines a card of information data with some items - Title, Image, Paragraph, and CTA Link. We are using the grid system here <code>assets/scss/grid.scss</code>. Molecules are used for creating this card.

<Story name="Image Reveal Cards">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return (
      <ImageRevealCards  
        data={caption.data}
        label={caption.label}
      ></ImageRevealCards>
    );
  }}
</Story>
