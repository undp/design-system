# Stats component

## Overview

The Stats Component is created for showing the Statspanel grid, designation, and image in particular section. It is a multi-usage component that creates are usually teasing some kind of content.

## Default Representation & various possible states

<Story name="Stats">
  {(args, { globals: { locale } }) => {
    const caption = statsArray(locale);
    return <Stats data={caption} imageback={BackgroundImg} ></Stats>;
  }}
</Story>
