### Usage
- Copy the HTML from the canvas
- Include the CSS and JS resource defined in the `Stats card`
- Include the `dist/css/components/stats-card-slider.min.css` file
- In you JS file initiliase following JS function
  - swiper('.stats-slider');
  - fontResize('.stats-card > h2');
- Add following JS files too
  - Swiper (https://swiperjs.com/)
  - dist/js/swiper.min.js
