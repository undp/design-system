};

# Table Tag component

The <code>table</code> HTML element represents tabular data — that is, the information presented in a two-dimensional table comprised of rows and columns of cells containing data. it's used to make large volumes of data easy to understand and access.

<ul>
<li>Table component created by <code>table</code> <code>tr</code> <code>th</code> and <code>td</code> tag.</li>
</ul>
