# Icons Component
Icons can grab attention and draw users into a piece of content. They can help users quickly understand the context in which they are placed.

## When to use
Use icons where the content is complex and dense to help users comprehend the available actions.
Icons can be used in place of long words or phrases, saving space.
Use Icons that are recognizable and accompany corresponding text labels where possible to make sure the users understand the context.

- Icons animation was done by the css
- Icons are saved in <code>assets/images/Icons</code> folder in SVG format.
