# Table

Tables are used to make large volumes of data easy to understand and access, by displaying data in tabular format.

We have applied different variations in table, which are as follows:
- Sizes: Table comes with large and small size
- Variant: default, striped and border
- Responsive: stacked, auto and scroll, fit for different use cases

### Usage
- Visit the **Canvas** tab
- Try different variations from the controls
- Go to HTML tab, copy the HTML and paste it in your HTML file
- Inlude **dist/css/base-minimal.css** in your HTML page, and you should have a styled table
