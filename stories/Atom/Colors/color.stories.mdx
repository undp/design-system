# Color Component
This component represents the various colors that can be used on the website as per brand guidelines. All these colors are defined as SCSS variables globally and used accordingly in all the components.

- Colors are divided into <code>Primary</code> and <code>Accents</code> colors.

#### 
#### 
### Primary Colors

Primary colors are the base colors of the UI, the ones that are used most frequently in the interface. They are used for brand elements and non-interactive elements. Interactive colors are used for primary links and action items that can be clicked or touched, to emphasize important actions and highlight information in your application.

### Accent Colors
Accent colors are colors that are used for emphasis in a color scheme. These colors can often be bold or vivid and are used sparingly, to emphasize, contrast or create rhythm. More toned down colors such as neutrals or darker shades can also be used depending on the surroundings.

#### Usage

- We can pick a color by HEX color code.
e.g., <code>#B5D5F5</code>. Colors are also defined in the variables by the same name like  <code>blue-100</code> it's <code>$color-blue-100</code>

  **use SCSS include the following files in your implementation**
- Include the common SCSS files like normalize, variables from <code>assets/scss</code>
- Component SCSS file: <code>assets/atom/colors/color.scss</code> 

  **To use CSS include the following files in your implementation**
- The compiled CSS file from <code>assets/css/style.css</code>
