# Breakpoints system

Desktop and Tablet
The grid system is working on the breakpoints, three breakpoint <code> small </code>  <code> medium </code> <code> Large </code>  are define to manage the layout according to devices resolutions with gutter spacing for Left or Right margin.

## Usage

- We can use grid by taking the grid class from the <code>assets/SCSS/grid.scss</code>
- <u>To use SCSS include the following files in your implementation</u>
- Include the common SCSS files like normalize, variables from <code>assets/scss</code>
- Component SCSS file: <code>layout.scss</code>
- <u>To use CSS include the following files in your implementation</u>
- The compiled CSS file from <code>assets/css/style.css</code>
