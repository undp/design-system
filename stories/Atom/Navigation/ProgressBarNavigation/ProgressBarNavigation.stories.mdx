# Progress Bar
The Progress Bar provides a visual representation of a user’s progress on page.

### Overview
A progress bar is a graphical content element which shows the progress of an ongoing task on a computer.

#### When to use:
- When the process requires more time to complete, so that the user gets the exact progress.

### Formatting
#### Default
The progress bar is an elongated rectangular bar which displays the progress of an extended computer operation.

### Behaviors

#### States
The progress bar has two states: default and short.

The following behavior modifiers are available:

**Default:**
The Yellow coloured rectangular bar with default states indicates the default progress bar which is used to display progress.

**Short:**
The Yellow colored rectangular bar with short states indicates the shorter progress bar which can be used instead of the default one to display progress.
