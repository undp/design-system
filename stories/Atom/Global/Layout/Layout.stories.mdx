# Grid system
The XY grid is the default Foundation grid, works very similarly to the standard float grid, but includes several useful features only possible with Flexbox, like horizontal and vertical alignment, automatic sizing and a full vertical grid.

<div className="subheading">Breakpoints</div>

Desktop and Tablet
The grid system is working on the breakpoints, three breakpoint <code> small </code>  <code> medium </code> <code> Large </code>  are define to manage the layout according to devices resolutions with gutter spacing for Left or Right margin.

The structure of XY grid uses classes <code>.grid-x</code>, <code>.grid-y</code>, and <code>.cell</code> as its base. Without defining a gutter type the cells will simply split up the space without any gutters.

<div className="subheading">Gutters</div>

The defining feature of the XY grid is the ability to use margin AND padding grids in harmony. To define a grid type, simply set  <code>.grid-margin-x</code>  / <code>.grid-margin-y</code> or <code>.grid-padding-x</code>/ <code>.grid-padding-y </code> on the grid.

<div className="subheading_2">Example</div>

<div className="usage_wrapper">
		<div className="grid-x grid-margin-x">
				<div className="cell  medium-6 large-6 grid--layout"></div>
				<div className="cell  medium-6 large-6 grid--layout"></div>
		</div>
    <div className="grid-x grid-margin-x">
        <div className="cell  medium-4 large-4 grid--layout"></div>
        <div className="cell  medium-4 large-4 grid--layout"></div>

		</div>
</div>

      <div class="cell medium-6 large-6"></div>

      <div class="cell medium-4 large-4"></div>
      <div class="cell medium-4 large-4"></div>
      <div class="cell medium-4 large-4"></div>
    </div>


<div className="subheading">Grid Container</div>

The grid defaults to the full width of the available space. To contain it horizontally use the <code>.grid-container</code> class. The container will be centered and have a max-width equal to your <code>$grid-container</code> setting (1440px by default), along with padding on the left/right equal to half your <code>$grid-container-padding</code> setting.

<div className="subheading_2">Example</div>

<div className="usage_wrapper"> 
	<div className="grid-container">
				<div className="grid-x grid-margin-x">
					<div className="cell small-4 grid--layout">	</div>
					<div className="cell small-4 grid--layout"> </div>
					<div className="cell small-4 grid--layout">	</div>
				</div>
		</div>
</div>

  <div class="grid-container">
    <div class="grid-x grid-margin-x">
        <div class="cell small-4"></div>
        <div class="cell small-4"></div>
        <div class="cell small-4"></div>
    </div>
  </div>

<div className="subheading">Grid Container Fluid</div>

To stretch the content to the full width of the available space, simply add the class <code>fluid</code> to your <code>grid-container</code>.

<div className="subheading_2">Example</div>

        <div class="cell small-4 grid--layout"></div>
      </div>

      </div>
    </div>

<div className="subheading">Auto Sizing</div>

If the class <code>.auto</code> or .<code>(size)-auto</code> is added to the cell, it will take up the remaining space.

A cell can also be made to shrink, by adding the .<code>shrink</code> or .<code>(size)-shrink</code> class. This means it will only take up the space its contents need.

<div className="subheading_2">Example</div>

  <div className="usage_wrapper"> 
    <div class="grid-x grid-margin-x">
      <div class="cell small-4 grid--layout"></div>
      <div class="cell auto grid--layout"></div>
      <div class="cell auto grid--layout"></div>
    </div>

  <div class="grid-x grid-margin-x">
      <div class="cell shrink grid--layout">shrink</div>
      <div class="cell auto grid--layout"></div>
    </div>
  </div>

    <div class="grid-x grid-margin-x">
      <div class="cell small-4"></div>
      <div class="cell auto"></div>
      <div class="cell auto"></div>
    </div>

    <div class="grid-x grid-margin-x">
      <div class="cell shrink"></div>
      <div class="cell auto"></div>
    </div>


<div className="subheading">Responsive Adjustments</div>

To switch back to the auto behavior from a percentage or shrink behavior, use the classes .<code>(size)-auto</code> or .<code>size-shrink</code>. In the below example, the cells stack on small screens, and become even-width on large screens.

  <div className="usage_wrapper">

      <div class="cell large-auto grid--layout"></div>
      <div class="cell large-auto grid--layout"></div>
      <div class="cell large-auto grid--layout"></div>
    </div>
  </div>

    </div>


<div className="subheading">Collapse Cells</div>

The .<code>(size)</code>-<code>gutter-type</code>-collapse class lets you remove cell gutters.

There are times when you won't want each media query to be collapsed. In this case, use the media query size you want and collapse and add that to your grid element. Example shows gutters at small and no gutters on medium and up.

<div className="subheading_2">Example</div>

  <div className="usage_wrapper"> 
    <div class="grid-x grid-margin-x medium-margin-collapse">
      <div class="cell small-6 grid--layout"></div>
      <div class="cell small-6 grid--layout"></div>
    </div>
  </div>

    <div class="grid-x grid-margin-x medium-margin-collapse">
      <div class="cell small-6"></div>
      <div class="cell small-6"></div>
    </div>

<div className="subheading">Offsets</div>

Offsets work by applying class <code>(size)-offset-2</code> margin-left to a grid.

<div className="subheading_2">Example</div>

  <div className="usage_wrapper"> 
    <div class="grid-x grid-margin-x">
      <div class="cell small-4 large-offset-2 grid--layout"></div>
      <div class="cell small-4 grid--layout"></div>
    </div>
  </div>

    <div class="grid-x grid-margin-x">
      <div class="cell small-4 large-offset-2"></div>
      <div class="cell small-4"></div>
    </div>

<div className="subheading">Block Grids <span>X Grid only</span></div>

To define cell widths within a direction-level, instead of the individual cell level, add the class .<code>(size)-up-(n)</code> to a <code>grid-x</code>, where <code>(n)</code> is the number of cells to display per direction, and <code>(size)</code> is the breakpoint at which to apply the effect.


<div className="subheading_2">Example</div>

  <div className="usage_wrapper">
     <div class="grid-x grid-padding-x small-up-2 medium-up-4 large-up-6">
      <div class="cell grid--layout"></div>
      <div class="cell grid--layout"></div>
      <div class="cell grid--layout"></div>
      <div class="cell grid--layout"></div>
      <div class="cell grid--layout"></div>
    </div>
  </div>

    <div class="grid-x grid-padding-x small-up-2 medium-up-4 large-up-6">
      <div class="cell">cell</div>
      <div class="cell">cell</div>
      <div class="cell">cell</div>
      <div class="cell">cell</div>
      <div class="cell">cell</div>
    </div>
