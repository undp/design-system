# Code Component
The code component is used to define a piece of computer code. The content inside is displayed in the browser's default monospace font.
<ul>
  <li>We can use the <code>code</code> in any component to render the text found within the <code>code tag</code> in monospace font.</li>
</ul>

Typographic scale when using code in text component:

<Story name="Code">
  {(args, { globals: { locale } }) => {

    return <p>{caption.detail1} <Code label={caption.detail2} ></Code> {caption.detail3} <Code label={caption.detail4} ></Code> {caption.detail5}</p>;
  }}
</Story>
