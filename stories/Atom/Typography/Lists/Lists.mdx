# List component
Lists consist of related items grouped together.

<div className="subheading">Unordered List</div>
<p>An unordered list is used if there is no hierarchy required. Unordered lists start with a bullet.</p>
<p>Start an unordered list with the <code>u1</code> tag.</p>
<p>Add a <code>li</code> tag for each list item</p>
<p>E.g</p>

<Story name="Unordered List">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <List data={caption} type="ul"></List>;
  }}
</Story>

<div className="subheading">Ordered List</div>
<p>An ordered list is used if there is a relation/hierarchy between list items. Ordered list items begin with a number generally.</p>
<p>Start an ordered list with the <code>o1</code> tag.</p>
<p>Add a <code>li</code> tag for each list item</p>

<Story name="Ordered List">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <List data={caption} type="ol"></List>;
  }}
</Story>

<div className="subheading">Description List</div>
<p>A description list is a list of terms, with a description of each term.</p>
<p>The <code>dl</code> tag defines the description list, the <code>dt</code> tag defines the term (name), and the <code>dd</code> tag describes each term.</p>

<Story name="Description List">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <Descriptionlist data={caption}></Descriptionlist>;
  }}
</Story>
