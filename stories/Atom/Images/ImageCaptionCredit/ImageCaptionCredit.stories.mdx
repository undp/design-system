# Image with caption and credit component

## Overview

The Image with caption and credit component multiusage component which creates boxes that are usually teasing some kind of content like - Image, Text, and Image author name. We are using a grid system to manage the different types of sizes and for the animation here assets/js/viewport.js. Image Caption Molecule are used for creating this card.
