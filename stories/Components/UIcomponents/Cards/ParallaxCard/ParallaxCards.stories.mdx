# Parallax Cards

The Parallax Cards component defines a card of information data with some items- Title, Paragraph, and CTA Link. 

### Overview
Parallax cards consist of textual and imagery sections. While scrolling through the page on desktops text part is scrolled with the normal speed and the image is “racing” along with greater velocity creating a parallax effect. Image can have a hover effect, sliding out the foreground picture and revealing the background image or video.

### When to use:

Can be used for the set of short bullet points accompanied with some imagery to provide an immersive reading experience for users

### Formatting

#### Default
The Parallax cards are usually used inside the grid to manage the different sizes and card orientations. It can also be accompanied with additional simple text elements 9title, description) and providing visual guide rails for individual cards.

On small screens, the widget collapses to the Carousel.

### Content

### Behaviors

#### States 

Default:
The Image is placed in either part of the grid and the Title, Paragraph, and CTA Link in the other part. As we scroll the page the image remains constant, while the contents move slightly up or down depending on the direction of the scrolling. The CTA link is expandable.

### Usage:
- Copy the HTML from HTML Tab and also include css and js files.
- Make sure to remove swiper-slider-0 swiper-initialized swiper-horizontal swiper-pointer-events remove these classes, remove ID and aria-label attributes as these are generated by the Swiper library
- Initialize the parallaxEffect by calling parallaxEffect('.parallax-card', ['.parallax-card__image', '.parallax-card__content'], 'top center', 'bottom+=85 center', 'vertical', 'desktop', 'percent') and swiper by calling swiper('.parallax__content') function on load.
- Include Swiper (https://swiperjs.com/get-started) and https://greensock.com/docs/v3/Installation GSAP libraries for parallax (include gsap, ScrollTrigger and EasePack) and slider in your html file.
- Refer [this document](https://github.com/undp/design-system/wiki/Swiper-documentation) for Swiper integration & options
- Refer [this document](https://github.com/undp/design-system/wiki/Parallax-documentation) for Parallax integration & options

- By clicking on the CTA link, the page is redirected to the detailed page.
