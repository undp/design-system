# Parallax Cards Component

## Overview

The Parallax Cards component defines a card of information data with some items - Title, Paragraph, and CTA Link. We are using a grid system to manage the different types of sizes. It is a multi-usage component that creates boxes that are usually teasing some kind of content.

## Default Representation & various possible states

<Canvas>
  <Story name="Parallax Cards">
    {(args, { globals: { locale } }) => {
      const caption = getCaptionForLocale(locale);
      return (
        <ParallaxCards alt={"feature card"} data={caption.data} title={caption.title} ></ParallaxCards>
      );
    }}
  </Story>
</Canvas>

## Dependencies

Libraries or another dependency which are used in the component.

- For animation functionality its depend on the jQuery plugin library.
- Custom JS file (SliderCustom.js) is located in the same js folder.
- Include JS Slider library file from <code>assets/js/swiper.js</code>
- Include JS Parallax library file from <code>assets/js/gsap.min.js</code>
- Include JS Parallax library file from <code>assets/js/ScrollTrigger.min.js</code>

## Usage

- Take HTML from the HTML tab in the canvas.
- <u>To use SCSS include the following files in your implementation</u>
- Include the common SCSS files like variables, mixin from <code>assets/scss</code>
- Component SCSS file: <code>parallaxcards.scss</code>
-  <u>To use CSS include the following files in your implementation</u>
- The compiled CSS file from <code>assets/css/style.css</code>
