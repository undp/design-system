# Stats Cards Component

## Overview

Stats Cards component are means of displaying content composed of different types of sizes. It is a multi-usage component that creates boxes that are usually teasing some kind of content.

we are using these modifier classes:
<code>.statscard--xl</code>, <code>.statscard--lg</code>, <code>.statscard--md</code>, <code>.statscard--sm</code>


## Default Representation & various possible states
