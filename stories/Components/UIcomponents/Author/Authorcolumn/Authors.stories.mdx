# Author
The author component gives an overview of the authors of an article/photo.

### Overview

The Author component highlights the author(s) ’s name, designation, and photo for an article or story they have contributed to on the site.

#### When to use:

- The author component is primarily used to give credit to any article or story they have published on the site.

### Formatting

#### Default
The author component consists of a single author or multiple authors.
If the author has a profile picture, it is displayed as follows,

<Canvas>
  <Story name="With picture">
    {(args, { globals: { locale } }) => {
      const caption = getCaptionForLocale(locale);
      return <Author image={user} alt='Headshot of XYZ' data={caption} {...args} Image="True"></Author>;
    }}
  </Story>
</Canvas>

If the author does not have a profile photo or there are 4 or more authors contributing to a single article/photo, then their information is displayed as follows.

<Canvas>
  <Story name="Without picture">
    {(args, { globals: { locale } }) => {
      const caption = getCaptionForLocale(locale);
      return <Author image={user} alt='Headshot of XYZ' data={caption} {...args} Image="False"></Author>;
    }}
  </Story>
</Canvas>

#### Multiple authors

If there are up to 3 authors, they are displayed horizontally.

<Canvas>
  <Story name="Multiple authors with picture">
    {(args, { globals: { locale } }) => {
      const caption = getCaptionForLocale(locale);
      return <Author Image="False" image={user} alt='Headshot of XYZ' data={caption} {...args} Number="Three"></Author>;
    }}
  </Story>
</Canvas>


If there are 4 or more authors, their profile pictures are not displayed.

<Canvas>
  <Story name="Multiple authors without picture">
    {(args, { globals: { locale } }) => {
      const caption = getCaptionForLocale(locale);
      return <Author Image="False" image={user} alt='Headshot of XYZ' data={caption} {...args} Number="Four" Image="False"></Author>;
    }}
  </Story>
</Canvas>
