# Author Card
An author card is a brief summary of an author highlighting their work.

### Overview
An author card gives a quick overview of an author. It includes their name, title, profile picture- optionally, a summary of their work, and a count of the number of articles they have published on the site. It acts as an entry point to all the articles published by them,

#### When to use:
- Author cards are mainly used in the author directory. 

### Formatting
#### Default
<Canvas>
  <Story name="Author Card">
    {(args, { globals: { locale } }) => {
      const caption = getCaptionForLocale(locale);
      return <Authorcard image= {user} alt= 'Headshot of XYZ' data={caption.authordata} para={caption.paragraph} button={caption.button} link='#' width="medium-12"></Authorcard>;
    }}
  </Story>
</Canvas>

### Content
An author card contains the first and last name, profile picture (optional), short summary, count of the number of articles published by them on the site, and a CTA to a more detailed description and list of their published articles.

### Behaviors
#### States
There is a single state for the Author card.
