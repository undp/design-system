# Lightbox Gallery

The Lightbox Gallery displays images in the front darkening the rest of the webpage.

### Overview
The Lightbox Gallery component when clicked on a particular image, pop-ups that specific image, gaining the users’ attention. 

#### When to use:

To gain the attention of the user.

### Formatting

#### Default
It consists of two sections: text with CTA link section and Image section.

### Content

### Behaviors

#### States 
There will be a set of 2-row images when clicked on a specific image then that image will be displayed in a popup view. The component is compatible with both mobile and tablet.

View of a set of images in a row: 

When selected a specific image:

<Canvas>
<Story name="Lightbox gallery">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <LightboxGallery 
        titleText={caption.titleText}
        descText={caption.descText}
        buttonText={caption.buttonText}
        data={caption.featureddata}
        {...args}
        ></LightboxGallery>;
  }}
</Story>
</Canvas>

### Usage:
- Copy HTML from the HTML tab of canvas and also include css and js files.
- Initialize lightbox gallery by calling lightbox_Gallery() function.
- Include below library files as well.
https://cdn.jsdelivr.net/npm/foundation-sites@6.7.4/dist/js/plugins/foundation.core.min.js
https://cdn.jsdelivr.net/npm/glightbox@3.1.0/dist/js/glightbox.min.js

### CSS and JS References

#### CSS:

Add the base layout style from dist/css/base-minimal.css
- dist/css/components/lightbox.min.css
- dist/css/components/lightbox-gallery.min.css
- dist/css/components/cta-link.min.css

JS:
- dist/js/lib/lightbox.min.js
- dist/js/lightbox-gallery.min.js

### Interactions
- By clicking on the image, it opens in the lightbox.
- By clicking on the close option, it can be closed.
- By swiping left/right or by using keyboard arrows, the images can be navigated.
