# Fixed image size carousel component

## Overview

The Fixed image size carousel Component is created for showing slider, designation, and image in particular section. It is a multi-usage component that creates are usually teasing some kind of content.

## Default Representation & various possible states

<Story name="Fixed Carousel">
  {(args, { globals: { locale } }) => {
    const caption = statsArray(locale);
    return <FixedSizeCarousel data={caption.data} label={caption.label}></FixedSizeCarousel>;
  }}
</Story>
