# Page Hero Component

## Overview

The Page hero component multiusage component which creates boxes that are usually teasing some kind of content like - Image, Text, Subtitle, and Breadcrumb. We are using the animation here assets/js/viewport.js. Atoms tags are used for creating this card.

## Default Representation & various possible states

#### Page hero - Full-width

<Story name="Page hero - Full-width">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return (
      <PageHero data={caption.herodata} title={caption.title} content={caption.content} active={'white'} imgsrc={caption.imgsrc} imgalt={caption.imgalt} {...args}
      ></PageHero>
    );
  }}
</Story>

### Page hero - Tall image

<Story name="Page hero - Tall image">

## Dependencies

Libraries or another dependency which are used in the component.

- For animation functionality its depend on the jQuery plugin library.
- Custom JS file (viewport.js) and (animation.js) is located in the same js folder.


## Usage

- Take HTML from the HTML tab in the canvas.
- <u>To use SCSS include the following files in your implementation</u>
- Include the common SCSS files like breakpoints, variables from <code>assets/scss</code>
-  Component SCSS file: <code>pagehero.scss</code>
- <u>To use CSS include the following files in your implementation</u>
- The compiled CSS file from <code>assets/css/style.css</code>
