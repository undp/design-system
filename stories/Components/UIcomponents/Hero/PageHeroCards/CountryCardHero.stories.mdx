# Page hero - Cards Component

## Overview

The Page hero - Cards component multiusage component which creates boxes that are usually teasing some kind of content like - Image, Text, Subtitle, and Breadcrumb. We are using the animation here assets/js/viewport.js. Atoms tags and Country Card Molecule are used for creating this card

## Default Representation & various possible states

<Story name="Page hero - Cards">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return (
      <CountryCardHero data={caption.countrydata} title={caption.title} subtitle={caption.subtitle}
      ></CountryCardHero>
    );
  }}
</Story>

## Dependencies

Libraries or another dependency which are used in the component.

- For animation functionality its depend on the jQuery plugin library.
- Custom JS file (viewport.js) is located in the same js folder.
- Custom JS file (SliderCustom.js) is located in the same js folder.

## Usage

- Take HTML from the HTML tab in the canvas.
- <u>To use SCSS include the following files in your implementation</u>
- Include the common SCSS files like breakpoints, variables from <code>assets/scss</code>
- Component SCSS file: <code>countrycardhero.scss</code> and <code>countrycard.scss</code>
- <u>To use CSS include the following files in your implementation</u>
- The compiled CSS file from <code>assets/css/style.css</code>
