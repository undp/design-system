# MultiSelect
The Dropdown allows the user to select one or multiple options.

### Overview
The Dropdown element allows users to choose single or multiple options from a list. It is used in forms when there are multiple choices that can be collapsed under one title.

#### When to use:
- When multiple options are to be grouped under a single title.

### Formatting
#### Default
It consists of a dropdown box with an arrow to indicate if it is closed or open.

#### Content
There are two types of Dropdown boxes: Standard and Multiselect.
1. Standard: In the standard dropdown box, only one option can be selected.
2. Multiselect: In the multiselect dropdown box, multiple options can be selected.
The direction of the arrow indicates if the box is opened or closed. The downward arrow indicates a closed box and upward arrow indicates open box.

### Behaviors
#### States
The Dropdown boxes are either opened or closed. 

**Standard:** The standard dropdown boxes either have auto or fixed height.

**Multiselect:** A select list where user can select multiple options from the dropdown.
