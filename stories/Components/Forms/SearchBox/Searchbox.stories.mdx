# Seach Box
A complete search widget for user to make search query.

### Overview
Input elements of type search are text fields designed for the user to enter search queries into.

#### When to use:
- When Search funtionality required

### Formatting

#### default
<Canvas>
  <Story name="Search-default">
    {(args, { globals: { locale } }) => {
      const caption = getCaptionForLocale(locale);
      return <Searchbox labelText={caption.detail3} type='search' id='search' mode='default-state' required='required' placeholder={caption.detail4}></Searchbox>;
    }}
  </Story>
</Canvas>

#### Error
<Canvas>
  <Story name="Search-error">
    {(args, { globals: { locale } }) => {
      const caption = getCaptionForLocale(locale);
      return <Searchbox labelText={caption.detail3} errorText={caption.detail2}  type='search' id='search' mode='error-state' required='required' placeholder={caption.detail4}></Searchbox>;
    }}
  </Story>
</Canvas>

#### Disabled
<Canvas>
  <Story name="Search Box">
    {(args, { globals: { locale } }) => {
      const caption = getCaptionForLocale(locale);
      return <Searchbox labelText={caption.detail3} errorText={caption.detail2} element='input' type='search' placeholder= {caption.detail4} {...args} ></Searchbox>;
    }}
  </Story>
</Canvas>

#### States
The Input Field has four states: Default, Focus. Error and Disabled.

### CSS and JS References
All input fields are part of default base style, no seperate CSS and JS required
