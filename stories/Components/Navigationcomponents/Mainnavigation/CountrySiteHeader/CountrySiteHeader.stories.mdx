# Country Header
The Country Site Header is a header which contains the site name .

### Overview
The Country Site Header component is a navigation bar which allows access to important functions across the website. It is present at the top of the page. 

#### When to use:
To indicate the location of the current page and to navigate back to each of the previously visited pages.

### Formatting
#### Default
It consists of a navigation bar with drop-down menus, CTA links,search bar, logo and site name. The Logo is situated at the left hand corner of the bar followed by the site title..

#### Content
There are two types of Country Header: Without Progress Bar and With Progress Bar

### Behaviors

#### States
There are three states:  Default, Tablet and Mobile.

  return <CountrySiteHeader siteTitleData={siteTitleData} backcaption={backcaption} locale={locale} leftNavigationData={leftNavigationData}
      navigationData={navigationData} rightNavigationData={rightNavigationData}
        menuData={menuData(locale)} languageswitcherData={Languageswitcher(locale)}
      text={caption}
        locationData={getLocationDataForLocale(locale)} langSelect={getSelectlanguage(locale)}
        ></CountrySiteHeader>;
  }}
