# Menu
The Menu bar is a horizontal bar with drop-down menus containing the links to other important pages of the website.

### Overview
The Horizontal menu consists of multiple clickable items placed at the top of the page. The navigation stays unchanged when browsing through the site and is present on every page.

#### When to use:
- When a drop-down list is to be created of important links.

### Formatting
#### Default
It consists of a horizontal bar with drop-down items.

### Content
The menu bar consists of a drop-down list for the important links.

### Behaviors

#### States
There are 3 states in Menu: Default, Hover and Open

1. Default: The Black coloured labels indicate the default state.
2. Hover: The Gray coloured labels indicate the hover state in which the label changes appearance when the mouse hovers over it.
3. Open: The Black coloured labels with a horizontal line below it indicate the open state.
