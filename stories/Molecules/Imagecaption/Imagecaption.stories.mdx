# Image Caption + Credit Component

## Overview

Image Caption + Credit component used to show the image caption and who posted content in a related page.

<div className="subheading">Modifier class</div>

<code>image__caption--unstag</code> <code>image__caption--stag</code>

<Canvas>
  <Story name="Image Caption" parameters={{ docs: { source: { code: `<figcaption class="image__caption">

      among nations.
    </div>
    <div class="image--author">
      <span class="author__caption author--label">UNDP/Angola</span>

    }}
