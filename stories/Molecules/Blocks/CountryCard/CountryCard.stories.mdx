# Country Card Component

## Overview

The country card component multiusage component which creates boxes that are usually teasing some kind of content like - Image, Text, and CTA Link. We are using the grid system here <code>assets/scss/grid.scss</code>. Atoms tags are used for creating this card.

<Story name="Country Card">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <CountryCard data={caption}></CountryCard>;
  }}
</Story>
