# Single Content Card Component
The Single Content Card component defines a card of information data with some items - Title, Image, Paragraph, and CTA Link. We are using the grid system here <code>assets/scss/grid.scss</code>. Atoms tags are used for creating this card.

<div className="subheading">Single Content Card</div>
<p>Single content card with image are show the Title, Image, and CTA Link. The background color is different on hover.</p>

<Story name="Single Content Card">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <SingleContent contenttile={caption.title} contentname={caption.name} contentnametwo={caption.nametwo}  paragraph={caption.paragraph} image={img} button={caption.btnlabel} link='#' {...args} ></SingleContent>;
  }}
</Story>
