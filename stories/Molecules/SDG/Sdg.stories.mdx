# SDG Component

The SDG Component is created for showing the title, image on hover. It is a multi-usage component that creates are usually teasing some kind of content.

## Default Representation & various possible states

<Story name="SDG">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return (
      <Sdg
        data={caption.data}
        imgsrc={sdglinkimage}
        title={caption.title}
      ></Sdg>
    );
  }}
</Story>

## Usage

- Take HTML from the HTML tab in canvas
- <u>To use SCSS include the following files in your implementation</u>
- Include the common SCSS files like normalize, variables, mixins from <code>assets/scss</code>
- Component SCSS file:<code>sdg.scss</code>
- <u>To use CSS include the following files in your implementation</u>
- The compiled CSS file from <code>assets/css/style.css</code>
