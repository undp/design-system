# Heading Big Block 

The Heading Block defines a section of information data with some items - Header title, Description. <code>H2</code> <code>H4</code> Atoms tags are used for creating this section.


<Story name="Heading Big Block">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <HeadingBig headerText={caption.header} descriptionText={caption.description} listData={caption.listData} {...args}></HeadingBig>;
  }}
</Story>
