# Author Card component

## Overview

The Author Card Component is created for showing the author name, designation, image, and content in particular section. It is a multi-usage component that creates are usually teasing some kind of content.

## Default Representation & various possible states

<div className="subheading">Author Card</div>

<Story name="Author Card">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <Authorcard data={caption.authordata} para={caption.paragraph} button={caption.button} link='#' width="12"></Authorcard>;
  }}
</Story>
