# Tab Component
Tab Component use to show the tabs to alternate between views within the same context, not to navigate to different areas. The active tab is the highlight with the red color.

<Story name="Tab">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <Tab tabdata={caption} ></Tab>;
  }}
</Story>
