# Feature Content Card Component

The Feature Content Card component defines a card of information data with some items - Tag, Title, Image, Paragraph, and CTA Link. We are using a grid system to manage the different types of sizes. It is a multi-usage component that creates boxes that are usually teasing some kind of content.

<div className="subheading">Feature Card With Image</div>

By default Feature content cards are show the information about the card with related fields like Tag, Title, Image, Description, and CTA Link. When we mouse over the card (On hover) it shows the image.

<div className="subheading">Feature Card Without Image</div>

When We mouse over the card (On hover) it shows the yellow color except the image.

<div className="subheading">Story Controls</div>

We can check all types of feature card variants by storybook <code> controls </code> .

<ul>
<li>the variant we check the option of card like <code>with image</code> and <code>without image</code></li>
<li>n the column, we check the option of card sizes like <code>large</code> , <code>medium</code> , <code>small</code> that's belong to the grid.</li>
</ul>

<div className="subheading2">Modifier class</div>

<code>with-image</code> <code>without-image</code> <code>feature__card--large</code> <code>feature__card--medium</code>
<code>feature__card--small</code> <code>large</code> <code>medium</code> <code>small</code>


<Canvas>
  <Story name="Parallax Card">
