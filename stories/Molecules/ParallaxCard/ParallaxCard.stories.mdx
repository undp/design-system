# Parallax Card Component

The Parallax Card component defines a card of information data with some items - Title, Paragraph, and CTA Link. We are using a grid system to manage the different types of sizes. It is a multi-usage component that creates boxes that are usually teasing some kind of content.

<Canvas>
  <Story name="Parallax Card">
    {(args, { globals: { locale } }) => {
      const caption = getCaptionForLocale(locale);
      return <ParallaxCard alt={'feature card'} data={caption}></ParallaxCard>;
    }}
  </Story>
</Canvas>
