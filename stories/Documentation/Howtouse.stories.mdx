# How to use our design system?
Using UNDP design system is very simples follow these steps and you will have exact component on your application.

## Steps to integrate
- Include the base styling CSS file in your project (`dist/css/base-minimal.css`).
- Browse the component which you want to integrate.
- Checkout the documentation page of the component.
- Copy the HTML of the component in your project.
- Checkout CSS and JS reference section in the document.
- Use the CSS and JS on your project as suggested.

## Development
If you are developing for this Design System, follow these steps and you will be ready for further development.

1. Checkout the codebase
```bash
git clone
```
2. Install project dependencies
```bash
npm install
```
3. Runs storybook locally
```bash
npm run storybook
```
4. Build storybook
```bash
npm run build-storybook
```
5. Linting on codebase
```bash
npm run lint
```
6. Run this command whenever compilinng CSS and JS
```bash
npm run build
```

## Dependencies

- Foundation Framework Only Flex Grid (https://get.foundation/)

- All the Icons can be found in the ***dist/images*** directory.
