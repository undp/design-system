# How to use our design system?
Using UNDP design system is very simples follow these steps and you will have exact component on your application.

## Usage
- Browse the component which you need
- Checkout the documentation page of the component
- Copy the HTML of the component
- Checkout CSS and JS reference of the component
- Use the CSS and JS on your project

## Development
If you are developing for this project, follow these steps and you will be ready for further development on this project.

1. Checkout the codebase
```bash
git clone
```
2. Install project dependencies
```bash
npm install
```
3. Runs storybook locally
```bash
npm run storybook
```
4. Build storybook
```bash
npm run build-storybook
```
5. Linting on codebase
```bash
npm run lint
```
6. Run this command whenever compilinng CSS and JS
```bash
npm run build
```
