# Font implementation

At the time of release of version 1.3.2, the UNDP Design System supports fonts for the following languages:

#### English, French, Spanish, Albanian, Bengali, Bosnia and Herzegovina, Montenegrin, Danish, German, Indonesian, Kazakh, Kyrgyz, Mongolian, Burmese, Norwegian, Portuguese, Romanian, Russian, Serbian, Finnish, Swedish, Turkish, Turkmenian, Ukrainian, Uzbek, Vietnamese.

Fonts used for these languages:

#### Proxima Nova, Sohne Breit, UNBangla

### In order to add Noto Sans fonts of languages that are not in the list above, click on the language you plan to use:

- Azerbaijani, Nepali ([Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans?query=noto+sans))
- Arabic, Kurdish ([Noto Sans Arabic](https://fonts.google.com/noto/specimen/Noto+Sans+Arabic?query=noto+sans+arabic))
- Chinese ([Noto Sans SC](https://fonts.google.com/noto/specimen/Noto+Sans+SC?query=noto+sans+sc))
- Japanese ([Noto Sans Japanese](https://fonts.google.com/noto/specimen/Noto+Sans+JP?query=noto+sans+Japanese))
- Georgian ([Noto Sans Georgian](https://fonts.google.com/noto/specimen/Noto+Sans+Georgian))
- Khmer ([Noto Sans Khmer](https://fonts.google.com/noto/specimen/Noto+Sans+Khmer))
- Korean ([Noto Sans Korean](https://fonts.google.com/noto/specimen/Noto+Sans+KR))

It will transfer to an external source (Google Fonts), follow all the instructions that are described there
