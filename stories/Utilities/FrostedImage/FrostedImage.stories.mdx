# Frosted background
The Frosted Background component is used to give a translucent effect to the background.

### Overview
The Frosted Background component is used to embed an overlay in a web page.

#### When to use:
To create an engaging background.

### Formatting
#### Default
It consists of a glassy background and an image on one side of the page.

### Behaviors
#### States
There is only one state: Default.
