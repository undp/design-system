# Filter and search
The Filter and Search option allows the user to apply filters to the search section.

### Overview
The filter and search option let’s the user filter the options to be selected. The sidebar component is a shorter piece of text that appears next to and accompanies a longer article.

#### When to use:
- When filters are to be added to the search bar.

### Formatting

#### Default
The search bar is rectangular shaped with a search icon. The active filters are chips with cross.

### Behaviors

#### State
- The search bar having a vertical line beside the search icon indicates that the search bar is in an open state.
- The search bar having a horizontal line below the search icon indicates that the search bar is in a close state.
- The sidebar component with a red coloured box and empty space in default state indicates that the option is not selected. 
- The sidebar component with a red coloured box and empty space in hover state indicates that the option is not selected and it changes appearance once the mouse hovers over it.
- The sidebar component with red coloured box and red tick in default state indicates that the option is selected.
- The sidebar component with red coloured box and red tick in hover state indicates that the option is selected and it changes appearance once the mouse hovers over it.
- The sidebar component with download icon in default state indicates that the option can be downloaded. 
- The sidebar component with download icon in hover state indicates that the option can be downloaded and it changes appearance once the mouse hovers over it.
- The sidebar component with external link icon in default state indicates that the option contains an external link related to the search.
- The sidebar component with external link icon in hover state indicates that the options contains an external link related to the search and it changes appearance once the mouse hovers over it.
