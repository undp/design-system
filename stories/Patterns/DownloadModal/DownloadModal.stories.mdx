# Download Modal component

## Overview

The Download Modal Component is created for showing the modal popup include: Image, Select, Listing and CTA Button. It is a multi-usage component that creates are usually teasing some kind of content.

## Default Representation & various possible states

<Story name="Download Modal">
  {(args, { globals: { locale } }) => {
    const caption = statsArray(locale);
    return <DownloadModal data={caption.data} button={caption.button} select={caption.select} {...args}></DownloadModal>;
  }}
</Story>

## Dependencies

Libraries or another dependency which are used in the component.

- For animation functionality its depend on the jQuery plugin library.
- Custom JS file (downloadmodal.js) is located in the same js folder.

## Usage

- Take HTML from the HTML tab in the canvas.
- <u>To use SCSS include the following files in your implementation</u>
- Include the common SCSS files like breakpoints, variables from <code>assets/scss</code>
- Component SCSS file: <code>downloadmodal.scss</code>
- <u>To use CSS include the following files in your implementation</u>
- The compiled CSS file from <code>assets/css/style.css</code>
