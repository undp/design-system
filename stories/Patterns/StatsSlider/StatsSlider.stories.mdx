# StatsSlider component

## Overview

The StatsSlider Component is created for showing the card grid in desktop with parallax effects and slider in mobile view, designation, and image in particular section. It is a multi-usage component that creates are usually teasing some kind of content.

## Default Representation & various possible states

<Story name="Stats slider (two rows)">
  {(args, { globals: { locale } }) => {
    const caption = statsArray(locale);
    return <StatsSlider data={caption}></StatsSlider>;
  }}
</Story>
